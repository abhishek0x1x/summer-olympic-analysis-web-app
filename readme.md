# 🏅 Olympics Data Analysis Web App

A complete data analysis and interactive visualization project based on 120 years of historical Olympics data. This project combines data analysis techniques with an interactive Streamlit web app for exploration and insights.

---

## 🚀 Live Demo

👉 [https://olympics0x1x.streamlit.app/](https://olympics0x1x.streamlit.app/)

---

## 📌 Project Overview

This project analyzes historical Olympic data to uncover patterns, trends, and insights such as:

* Medal distribution across countries
* Performance trends over years
* Athlete participation
* Country-wise dominance

The analysis is implemented in a Jupyter Notebook and deployed as an interactive web application using Streamlit.

---

## 🛠️ Tech Stack
